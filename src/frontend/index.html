Transform Topics into Educational Videos

AI-powered video generation with Manim animations. Enter a topic or paste your own lecture script — from 60-second explainers to 30-minute long-form lessons.

Recommended format: Scene N: Title, then Narration: and Visual: on separate lines. Also accepts Voiceover/Animation/On-screen labels, markdown headings, [VISUAL: ...], or JSON (use JSON only mode).
